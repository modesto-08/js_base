# 🧠 JavaScript - Exercícios Práticos  

## 💻 Sobre o Repositório

Este é um repositório criado com o objetivo de armazenar e organizar atividades desenvolvidas para praticar os principais conceitos da linguagem JavaScript.

Os exercícios abordam desde fundamentos básicos até manipulação do DOM, lógica de programação e interatividade com páginas web.

Este repositório faz parte das atividades desenvolvidas durante o curso **Técnico em Desenvolvimento de Sistemas** do SENAI.

---

## 🎯 Objetivo

Praticar lógica de programação.
Aprender e aplicar conceitos fundamentais do JavaScript.
Desenvolver raciocínio para resolução de problemas.
Manipular elementos HTML através do DOM.
Criar interações dinâmicas em páginas web.

---

## ⚙️ Conceitos Aplicados

Durante os exercícios, foram trabalhados os seguintes conceitos:

✅ Variáveis e Tipos de Dados
✅ Operadores Lógicos e Aritméticos
✅ Estruturas Condicionais (if, else, switch)
✅ Estruturas de Repetição (for, while)
✅ Funções
✅ Arrays
✅ Eventos no DOM
✅ Manipulação de Elementos com JavaScript
✅ Validação de Formulários
✅ Boas práticas de organização de código
